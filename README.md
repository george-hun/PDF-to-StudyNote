# PDF Note Generator (굿노트 최적화 PDF 변환기)

이 프로젝트는 강의 슬라이드 PDF를 굿노트(GoodNotes)에서 필기하기 좋은 형태로 변환해주는 도구입니다. 한 페이지에 두 개의 슬라이드를 배치하고 오른쪽에 필기용 모눈종이 공간을 추가하여 학습 효율을 높여줍니다!

## ✨ 주요 기능

- **굿노트 최적화**: 1440 x 1080 규격으로 생성되어 아이패드/태블릿에서 필기하기에 가장 적합합니다.
- **2-슬라이드 세로 배치**: 왼쪽 절반에 슬라이드 2개를 세로로 배치합니다.
- **필기용 모눈종이 (Grid)**: 오른쪽 절반에 깔끔한 모눈종이 영역을 생성합니다.
- **자동 라이브러리 설치**: 프로그램을 실행하면 필요한 라이브러리(`PyMuPDF`)가 없을 경우 자동으로 설치를 진행합니다.
- **역방향 회전 상쇄**: 슬라이드가 회전된 상태인 경우에도 이를 감지하고 보정하여 올바른 방향으로 출력합니다.

## 🚀 사용 방법

1.  **준비**: 프로젝트 폴더 안에 `input_pdf` 폴더를 만들고 변환할 PDF 파일들을 넣습니다. (`output_pdf` 폴더도 자동 생성됩니다.)
2.  **실행**: 터미널 또는 명령 프롬프트에서 아래 명령어를 실행합니다.

    ```bash
    python makepdf.py
    ```

    *※ 라이브러리가 없다면 첫 실행 시 자동으로 설치되므로 잠시 기다려주세요.*

3.  **확인**: 변환된 파일은 `output_pdf` 폴더에서 확인할 수 있습니다.

## 📂 파일 설명

-   **`makepdf.py` (권장)**: 가장 성능이 좋고 편리한 메인 스크립트입니다. 라이브러리 자동 설치 기능과 스마트 회전 보정 기능이 포함되어 있습니다.
-   **`안될시 이거 사용.py`**: 위 파일로 변환했을 때 내용이 깨지거나 레이아웃이 비정상적일 때 사용하는 예비용 스크립트입니다. 이미지를 렌더링하는 방식이라 더 안정적이지만, 변환 속도가 느리고 파일 용량이 커질 수 있습니다.

## 📋 요구 사항

- Python 3.x 이상
- 인터넷 연결 (첫 실행 시 라이브러리 자동 설치를 위해 필요)

## ⚠️ 참고 사항

- 입력 PDF 파일은 반드시 `input_pdf` 폴더 안에 있어야 합니다.
- 변환 중 문제가 발생하면 `안될시 이거 사용.py`를 활용해 보세요.
